{% endif %} Reminder of Receiver's Payoffs
You get the highest payoff for guessing correctly about the drawn set.
If the drawn set is Up, you get $15 for guessing Up, $12 for guessing Middle, and $3 for guessing Down.
If the drawn set is Middle, you get $15 for guessing Middle, $8 for guessing Up and $12 for guessing Down.
If the drawn set is Down, you get $15 for guessing Down, $8 for guessing Middle and $3 for guessing Up.

If you are uncertain about what the drawn set is:
If you think the drawn set is equally likely to be Middle or Down, guessing Down leads to the highest expected payoff.
If you think the drawn set is equally likely to be Up or Middle, guessing Middle leads to the highest expected payoff.
If you think the drawn set is equally likely to be Up, Middle or Down, guessing Middle leads to the highest expected payoff.
